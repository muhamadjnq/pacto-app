# Pacto — پروژه‌ی اپ اندروید

## ✅ روش پیشنهادی: ساخت APK با گیت‌هاب (بدون نصب چیزی)
فایل **`راهنما-گیت‌هاب.md`** را باز کن و قدم‌به‌قدم برو.
این روش APK را روی سرور گیت‌هاب می‌سازد — نه جاوا، نه SDK، نه فیلترشکن روی سیستم تو.

## روش جایگزین: ساخت روی کامپیوتر خودت
اگر خواستی روی سیستم خودت بسازی، فایل **`راهنما-ویندوز.md`** را ببین
(به نصب Node.js، JDK و Android Studio نیاز دارد).

## محتوای این پوشه
- `www/index.html` — خود اپ Pacto
- `.github/workflows/build-apk.yml` — فایلی که گیت‌هاب با آن خودکار APK می‌سازد (مهم!)
- `make-offline.js` — آفلاین‌کردن اپ
- `capacitor.config.json` ، `package.json` — تنظیمات پروژه
- `resources/icon.svg` — آیکون اپ
- `راهنما-گیت‌هاب.md` — راهنمای روش گیت‌هاب (پیشنهادی)
- `راهنما-ویندوز.md` — راهنمای ساخت روی ویندوز
